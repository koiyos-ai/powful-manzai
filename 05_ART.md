# 05 ART ─ キャラクター画像制作ガイド

> **分割仕様書のうち「絵の作り方」を扱う制作ガイド。**
> 全体像は `00_OVERVIEW.md`。キャラの性格・設定は `04_STORY.md` を参照。
> 画像はゲーム本体(HTML)に組み込み、紙芝居方式でシーンを表現する。

---

## 1. 基本方針 ✅

- **表現方式**: 紙芝居方式。**立ち絵 + 表情差分 + テキスト**でシーン・エピソードを表現する(パワプロのサクセス紙芝居と同方式)。
- **画像生成ツール**: **ChatGPT(GPT-4o画像生成)** で生成する。**Claudeは画像生成不可**のため、Claudeの役割はプロンプト設計のみ。
- **漫才プレイ中のキャラとの関係**: サクセス/シナリオの立ち絵と、漫才リズムゲーム中のSVG2頭身キャラは**別物でよい**(初期パワプロもサクセス紙芝居と試合中で絵が異なっていた)。統一感は必須としない。

---

## 2. 共通スタイル基盤 ✅
全キャラを同一テイストに揃えるため、毎回プロンプト冒頭に付ける共通文。

```
2頭身のデフォルメキャラクター。野球ゲーム「実況パワフルプロ野球」のパワプロくんのような、頭が大きく丸い、可愛くポップなアニメ調。太い輪郭線、明るくクリアなセル塗り、シンプルで親しみやすいデザイン。日本の家庭用ゲームのキャラクターアート。背景は透明(透過PNG)。キャラクター全身を正面から、棒立ちのニュートラルポーズで。
```

---

## 3. 一貫性を保つ手順 ✅(重要)
立ち絵と表情差分で別人になると紙芝居が破綻するため、以下を厳守。

1. まず**基準立ち絵を1枚確定**させる(納得いくまで調整)。
2. 表情差分は、**その基準画像を添付した上で**、同じ会話の続きで「この画像のキャラの表情だけ変える」と指示して作る。
3. 顔がブレたら、基準画像を**再添付**して「この顔を厳密に保って」と念押しする。
4. 1キャラ分は1つの会話の中で続けて作る(会話を分けると別人化しやすい)。
5. 背景は常に透過PNGを指定。

---

## 4. キャラ別プロンプト

### 4.1 主人公 ✅ 基準立ち絵=確定済み
**確定したデザイン**: 赤いスーツ + 黄色いネクタイ、ツンと跳ねた黒髪、太く力強い眉、鋭くも芯のある目、ほっぺの赤み、白い手袋(豆のような手)、茶色い靴。崖っぷち7年目の負けん気と、どこかの哀愁を表現。鉢巻きは無しで確定。

> 基準立ち絵はChatGPTで生成済み・採用済み。以降は下記プロンプトで**表情差分のみ**を作る。

**基準立ち絵プロンプト(記録用・再生成が必要な場合)**:
```
(共通スタイル基盤)
キャラクター:崖っぷちの漫才師、芸歴7年目の男性芸人。30代手前。
赤いスーツに黄色いネクタイ。髪はツンと跳ねた短い黒髪。きりっとした太い眉。
芯の強さと負けん気を感じる顔立ちだが、どこか哀愁もある。
表情はニュートラル(普通の真顔)。ほっぺに小さな赤み。
白い手袋、茶色い靴。全身、正面、棒立ち。透過背景。
```

### 4.2 相方:矢田くん ✅(設定確定 / 立ち絵は未生成)
**設定**(詳細は `04_STORY.md`): 主人公と同い年の相方。青いスーツ + 黄ネクタイ。やや顎が長め(しゃくれ気味)の輪郭で親しみやすく情けない顔。落ち着いた七三分けの黒髪。垂れ眉とタレ目。気弱だが憎めないムードメーカー。パワプロの矢部くん的な飄々とした雰囲気。

**基準立ち絵プロンプト**:
```
(共通スタイル基盤)
キャラクター:お笑いコンビの相方の男性芸人。主人公と同い年。
青いスーツに黄色いネクタイ。やや顎が長め(しゃくれ気味)の輪郭で、親しみやすく情けない顔立ち。
落ち着いた七三分けの短い黒髪。垂れ眉とタレ目で、気弱だが憎めない雰囲気。人懐っこいムードメーカー。
表情はニュートラル(人の良さそうな普通の表情)。ほっぺに小さな赤み。全身、正面、棒立ち。透過背景。
```
> 生成時のコツ: 主人公の基準立ち絵も一緒に添付し「この子と同じ画風・同じ2頭身バランスで」と指示してテイストを揃える。

## 5. 必要な表情差分(紙芝居用) 🔧
各キャラ(特に主人公・矢田)について、以下の表情を基準立ち絵から派生生成する。

| 表情 | 用途 | 差分指示の例 |
|------|------|------------|
| 通常 | 基本(基準立ち絵そのもの) | ― |
| 笑顔 | 日常の明るい場面 | 口角を上げた明るい笑顔。目もにこやか |
| 大笑い | 最高にウケた時 | 目を閉じて口を大きく開けた大爆笑 |
| 悔しい | 賞レース敗退など | 歯を食いしばり眉を寄せ、目に悔し涙 |
| 驚き | 想定外の展開 | 目を大きく見開き口を開ける。汗マーク |
| 落ち込み(スベリ) | ネタがスベった時 | うつむき、青ざめた暗い表情。顔に青い縦線 |
| 真剣(気合い) | 本番前の集中 | 眉に力を入れた真剣な眼差し、口を引き結ぶ |
| 怒り | 矢田との喧嘩など | 眉を吊り上げ口を開けて怒る |

**表情差分の共通指示文**:
```
この立ち絵のキャラクターの顔・髪型・服装・体型・画風・透過背景はそのまま厳密に保ち、表情だけを変えてください。同じ正面・棒立ちポーズ。表情:【上表から1つ】
```

---

## 6. HTML組み込みメモ 💡
- 立ち絵はキャラ単位で1枚、表情差分は同寸・同位置で書き出すと、HTML側で画像src差し替えのみで表情を切り替えられる。
- 透過PNGなら背景レイヤー(劇場・河川敷など)の上に重ねられる。
- ファイル命名例: `hero_normal.png` `hero_smile.png` `hero_kuyashii.png` … キャラ名_表情。
