## 2. キャラクター育成パラメータ ✅

### 2.1 基礎能力(5種)
最終的にキャラに固定される、パワプロの「ミート・パワー」等に相当する能力。
**ネタの形式(しゃべくり/コント)によって、どの能力が活きるかが変わる**(§3参照)。

| 能力 | 内容 | 役割 |
|------|------|------|
| **ボケ** | ボケの威力・面白さ | **共通**(両形式で常に参照) |
| **ツッコミ** | ボケを笑いに変換する力 | **共通**(両形式で常に参照) |
| **大喜利** | 瞬発的な発想力・切り返し | **共通**(両形式で常に参照。コントでも必要) |
| **トーク** | フリートーク・喋りの力 | **しゃべくり漫才で活きる**(コントでは寄与小) |
| **演技力** | 役の没入感・キャラの説得力 | **コント漫才で活きる**(しゃべくりでは寄与小) |

- 評価表記は **G〜S のグレード**(パワプロ準拠)。内部値は 0〜100。
- グレード境界: S(90+) / A(80+) / B(70+) / C(60+) / D(50+) / E(50未満)。
- **能力体系の要点**: ボケ・ツッコミ・大喜利は漫才の普遍的基礎力でどちらの形式でも使う。そこに、しゃべくりなら**トーク**、コントなら**演技力**が加わる。これにより、しゃべくり特化ビルド(トーク重視)とコント特化ビルド(演技重視)という育成方針の分岐が生まれ、全能力に活躍の場がある。
- ※各パーツの性質タグごとに「どの能力をどの比率で参照するか」が定義される。具体的な配分テーブルと得点計算は `SCORING.md` で扱う。

### 2.2 サクセス中のリソースステータス(2種)
育成プレイ中のみ使う、消費・回復するリソース。

| ステータス | 内容 |
|------------|------|
| **体力** | 行動の燃料。本家パワプロと同様の挙動。コマンドで消費、休む/遊びで回復。 |
| **発想力**(オリジナル要素) | ネタ作りに影響。**高いほどネタ作りでレアパーツが出やすい**。インプットで貯め、ネタ作りで消費する「クリエイティブの貯金」。 |

### 2.3 特殊能力(青特/金特/赤特) ✅(体系) / 🔧(全リスト)
パワプロの金特・赤特に相当。本番でエフェクトを発動する。

- **青特(プラス小)**: ◯つかみ / ◯天丼 / ◯スベリ耐性 / ◯客いじり / ◯トップバッター / ◯審査員ウケ / ◯生放送
- **金特(プラス大)**: ◎会場ジャック(爆笑の連鎖) / ◎伝説のネタ(ネタ評価上限突破) / ◎しゃべくりの神 / ◎一心同体(相方とのミスフォロー。§9.3 高信頼度ルートのイベントで取得)

> **◯天丼について**: 「天丼」(同じボケを後半で再度=かぶせ)は性質タグではなく**コンボ/特能側**で扱う(§3.5)。同じ威力・性質のボケが1ネタに2つあると成立する弱コンボ(小技)を基本とし、青特◯天丼はその威力・成立を強化する。実際の漫才でも天丼は単体ボケの質ではなく2つのボケの関係で成立するため、コンボとして表現する。
> **ボケツッコミについて**: 「ボケで返すツッコミ」も単体の質タグではなく、ボケとツッコミの境界を溶かす上級技としてコンボ/特能側で扱う(質タグからは除外)。

> **◎一心同体について**: 本番でミスをフォローする効果は強力なため、通常の青特(◯コンビ愛)ではなく**金特**として位置づけ、§9.3の解散危機・高信頼度ルートの友情イベントでのみ取得可能とする。効果は「本番のミスを複数回フォロー」または「温度が低い時に発動して立て直す」方向で今後数値化(🔧)。
- **赤特(マイナス)**: ✕テンパり / ✕時間オーバー / ✕単独運✕ / ✕審査員と犬猿 / ✕酒癖

> 実装方式は「能力フックシステム」を採用(§4.6 参照)。

## 3. 漫才ネタ構成システム(打線を組む) ✅

### 3.1 基本思想
パーツの中身は**テキスト化しない**(実際の漫才に面白さで敵わない & 会話成立が困難なため)。
代わりに **抽象表現**を用いる。

- 表記例: `ボケ【6】(王道)` `ツッコミ【9】(どつき)`
- 各パーツは「**種別** + **威力数値【1〜9】** + **性質タグ**」で表現。
- 威力は **1〜9**(基礎能力の G〜S 表記と視覚的に区別するため、あえて数字)。

### 3.2 パーツの取得
- 「**ネタ作り**」コマンドやイベントで漫才パーツを入手。
- その時の**発想力が高いほどレアパーツ(高威力・希少タグ)が出やすい**。
- 発想力はネタ作りで消費される。

### 3.3 ネタ構成(打線)
- **9スロット制**(パワプロの打線=9人に準拠)。
- **1スロット目=ツカミ固定、9スロット目=オチ固定**。2〜8番が自由枠。
- 威力数値の合計+コンボボーナスを「**手応え**」として表示(プレイヤーに見える概算期待値。詳細は `SCORING.md`)。
- ※ただし「手応えがあってもウケない」現象を再現するため、本番結果には会場・出順・審査員補正が乗る。

#### 3.3.1 ネタの形式(しゃべくり漫才 / コント漫才) ✅
**ネタを組む際、まず形式を「しゃべくり漫才」か「コント漫才」のどちらかに決める**(パワプロの野手/投手のような大別)。

- **完全分離**: 1つのネタ(打線)は、どちらかの形式で**統一**する。しゃべくり用パーツとコント用パーツは**混在できない**。
- パーツは形式ごとに別物として存在する(例:「しゃべくり用の あるあるボケ」と「コント用の あるあるボケ」は別パーツ)。ネタ作りで集める段階から形式が区別される=ハクスラ的な収集の二系統。
- **形式による能力の活き方**(§2.1):
  - **しゃべくり漫才** → ボケ/ツッコミ/大喜利(共通) + **トーク**(喋りの力が活きる)
  - **コント漫才** → ボケ/ツッコミ/大喜利(共通) + **演技力**(演じる力が活きる)
- これにより、しゃべくり特化ビルド / コント特化ビルドという**構築の二大方針**が生まれる。共通能力(ボケ/ツッコミ/大喜利)は両方で育つので、地続きの成長もある。
- **段階解放(予定)**: ゲーム序盤は**しゃべくり漫才のみ**で進め、ストーリー途中の強制イベントで**コント漫才をアンロック**する方針(初心者の認知負荷を下げ、実装も段階的にするため)。🔧要検討。
- 配置したパーツの性質タグの比率で、ネタの**ジャンルを自動判定**(【王道しゃべくり】【勢いコント】等。形式×タグ傾向で命名)。

### 3.4 パーツ種別と性質タグ ✅
実際の漫才理論(ボケ18種・ツッコミ12種等)を土台に、ゲーム用に厳選。
**形式分離**の有無は種別ごとに異なる(打順固定のツカミ・オチは形式共有、中間スロットのフリ・ボケ・ツッコミはしゃべくり/コント分離)。

#### ツカミ(1番固定・2種・形式共有)
| タグ | 性質 |
|------|------|
| 自己紹介 | 「どうも〜」と名乗って入る王道の掴み。安定型(トーク寄り) |
| ギャグ | 開始直後に一発ギャグをかまして掴む。笑い先行型(ボケ寄り) |

#### フリ(中間・4種・しゃべくり/コント分離)
喋りで状況・話題を作り、ボケの前振り(緊張をためる)をする役割。
| タグ | 性質 |
|------|------|
| 日常 | 「〜ってあるよね」共感ベースの自然な前振り。あるあるボケと好相性(トーク寄り) |
| 時事 | 旬の話題・トレンドから入る。共感・旬(トーク寄り) |
| 説明 | これから起きることを段取り良く説明し土台を作る。伏線設計・構成型(大喜利寄り) |
| 持論 | 「俺はこう思う」独自の主張・理論を展開。知性・しゃべくり型(大喜利寄り) |

#### ボケ(中間・5種・しゃべくり/コント分離)
| タグ | 性質 |
|------|------|
| あるある | 現実にありそうな共感系。事故りにくい安定型。日常フリと好相性 |
| 誇張 | 大げさ・過剰・行き過ぎ・暴走。パワー型。ハマると倍率大 |
| 裏切り | 期待を外す・スカシ・逆を言う。意外性型。スカシツッコミと好相性 |
| 動き | 体・顔を使うボケ。視覚型。コント形式で特に活きる(演技寄り) |
| ひねり | 言葉遊び・韻・もじり・言い回しの妙。知性型(大喜利・トーク寄り) |

#### ツッコミ(中間・5種・しゃべくり/コント分離)
| タグ | 性質 |
|------|------|
| 直球 | 「なんでやねん」系の王道ストレート。どつき的な強さも含む基本型。安定 |
| 例え | ボケを何かに例えて突っ込む。知性型。大喜利の影響大(「生まれたての子鹿か」系) |
| ノリ | 一度乗ってから突っ込む。テンポ・演技型。直前のボケの笑いを増幅 |
| 説明 | 理詰めで正す・一刀両断。構成・知性型(大喜利寄り) |
| スカシ | 強く突っ込まず流す・はぐらかす。クール・技巧型。裏切りボケと好相性 |

#### オチ(9番固定・2種・形式共有)
| タグ | 性質 |
|------|------|
| 回収 | 前半のフリ・要素を回収して締める。技巧・構成型(大喜利+ボケ寄り)。技巧派審査員に刺さる |
| 裏切り | 最後に予想を裏切る・大きくひっくり返して締める。インパクト型(ボケ寄り)。勢い/新規性派に刺さる |

> **設計メモ**:
> - 各タグは「どの能力をどの比率で参照するか(合計10)」を持つ。共通能力(ボケ/ツッコミ/大喜利)＋形式固有(しゃべくり=トーク/コント=演技)。具体的な配分テーブルと得点計算は `SCORING.md` で定義。
> - 「天丼」「ボケツッコミ」は性質タグではなく**コンボ/特能側**で扱う(§2.3, §3.5)。
> - フリ・ボケ・ツッコミは**しゃべくり版/コント版**を別パーツとして用意。ツカミ・オチは打順固定のため**形式共有**(1タグにつき1つ)。

### 3.5 コンボ(役満)システム ✅
特定のパーツ配置で発動する追加得点ボーナス。

- **3段階のレア度**: 小技 / 大技 / 役満。
- **天丼コンボ(基本の小技)** ✅: 同じ威力・同じ性質タグのボケが1ネタに2つあると成立(発生しやすい弱コンボ)。前半のボケを後半でかぶせる「天丼」を表現。青特◯天丼で威力・成立が強化される。最初に発見しやすい入門コンボ。
- 役満・大技コンボの例(新タグ体系):
  - 【怒涛の畳みかけ】: 誇張ボケ + 直球ツッコミ の交互4連を6〜8番に配置(勢い型)
  - 【全回収】: 説明フリ2箇所以上 + 回収オチ(伏線を張って回収する技巧型)
  - 【裏切りの応酬】: 裏切りボケ + スカシツッコミ の連続(意外性型)
  - 【完全天丼】: 同じボケを間隔をあけて3箇所に配置(天丼の上位・様式美)
  - 【教科書】: あるある/直球など王道タグのみで構成(低倍率だが事故率ゼロ。王道派審査員向き)
  - 【コント名人】: コント形式で動きボケ + ノリツッコミ主体(演技型)
- 小技コンボは2〜3パーツの部分役(例:日常フリ→あるあるボケの黄金の入り、ノリツッコミ→次のボケ増幅)を多数散りばめる。
- ※コンボの具体的な得点ボーナス(加算か倍率か)は `SCORING.md` で今後設計。

### 3.6 コンボの習得と発見 ✅
コンボ発生順は難易度が高いため、**レシピを知る手段**を用意する。これが知識資産となり周回動機になる。

- **イベントで習得**: 「先輩からの助言」「大物作家からのアドバイス」で骨組みを教わる。相手の格でレシピの格が変わる(同期=小技 / 先輩=大技 / 大物作家・レジェンド=役満)。
- **偶然の発見**: 教わっていないコンボを偶然組んでも**発動する**。ネタ披露後、楽屋で「今日のネタ、〜〜みたいやな」と声をかけられコンボ名が発覚し、**一度披露すれば発見扱い**で図鑑登録。
