## 2. キャラクター育成パラメータ ✅

### 2.1 基礎能力(5種)
最終的にキャラに固定される、パワプロの「ミート・パワー」等に相当する能力。
**ネタの形式(しゃべくり/コント)によって、どの能力が活きるかが変わる**(§3参照)。

| 能力 | 内容 | 役割 |
|------|------|------|
| **ボケ** | ボケの威力・面白さ | **共通**(両形式で常に参照) |
| **ツッコミ** | ボケを笑いに変換する力 | **共通**(両形式で常に参照) |
| **大喜利** | 瞬発的な発想力・切り返し | **共通**(両形式で常に参照。コントでも必要) |
| **トーク** | フリートーク・喋りの力 | **しゃべくり漫才で活きる**(コントでは寄与小) |
| **演技力** | 役の没入感・キャラの説得力 | **コント漫才で活きる**(しゃべくりでは寄与小) |

- 評価表記は **G〜S のグレード**(パワプロ準拠)。内部値は 0〜100。
- グレード境界: S(90+) / A(80+) / B(70+) / C(60+) / D(50+) / E(50未満)。
- **能力体系の要点**: ボケ・ツッコミ・大喜利は漫才の普遍的基礎力でどちらの形式でも使う。そこに、しゃべくりなら**トーク**、コントなら**演技力**が加わる。これにより、しゃべくり特化ビルド(トーク重視)とコント特化ビルド(演技重視)という育成方針の分岐が生まれ、全能力に活躍の場がある。
- ※各パーツの性質タグごとに「どの能力をどの比率で参照するか」が定義される。具体的な配分テーブルと得点計算は `SCORING.md` で扱う。

### 2.2 サクセス中のリソースステータス(2種)
育成プレイ中のみ使う、消費・回復するリソース。

| ステータス | 内容 |
|------------|------|
| **体力** | 行動の燃料。本家パワプロと同様の挙動。コマンドで消費、休む/遊びで回復。 |
| **発想力**(オリジナル要素) | ネタ作りに影響。**高いほどネタ作りでレアパーツが出やすい**。インプットで貯め、ネタ作りで消費する「クリエイティブの貯金」。 |

### 2.3 特殊能力(青特/金特/赤特) ✅(体系) / 🔧(全リスト)
パワプロの金特・赤特に相当。本番でエフェクトを発動する。

- **青特(プラス小)**: ◯つかみ / ◯天丼 / ◯スベリ耐性 / ◯客いじり / ◯トップバッター / ◯審査員ウケ / ◯生放送
- **金特(プラス大)**: ◎会場ジャック(爆笑の連鎖) / ◎伝説のネタ(ネタ評価上限突破) / ◎しゃべくりの神 / ◎一心同体(相方とのミスフォロー。§9.3 高信頼度ルートのイベントで取得)

> **◯天丼について**: 「天丼」(同じボケを後半で再度=かぶせ)は性質タグではなく**コンボ/特能側**で扱う(§3.5)。同じ威力・性質のボケが1ネタに2つあると成立する弱コンボ(小技)を基本とし、青特◯天丼はその威力・成立を強化する。実際の漫才でも天丼は単体ボケの質ではなく2つのボケの関係で成立するため、コンボとして表現する。
> **◯時事について**: 「時事ネタ」はフリの通常タグから除外し、青特**◯時事**として扱う(🔧効果は今後設計)。時事ネタはしゃべくり的な行為で、コント(役を演じる)の中では素に戻り不自然になるため、形式分離の枠組みに乗らない特殊な要素。特能保持者のみが扱える持ち味とする。
> **ボケツッコミについて**: 「ボケで返すツッコミ」も単体の質タグではなく、ボケとツッコミの境界を溶かす上級技としてコンボ/特能側で扱う(質タグからは除外)。

> **◎一心同体について**: 本番でミスをフォローする効果は強力なため、通常の青特(◯コンビ愛)ではなく**金特**として位置づけ、§9.3の解散危機・高信頼度ルートの友情イベントでのみ取得可能とする。効果は「本番のミスを複数回フォロー」または「温度が低い時に発動して立て直す」方向で今後数値化(🔧)。
- **赤特(マイナス)**: ✕テンパり / ✕時間オーバー / ✕単独運✕ / ✕審査員と犬猿 / ✕酒癖

> 実装方式は「能力フックシステム」を採用(§4.6 参照)。

## 3. 漫才ネタ構成システム(打線を組む) ✅

### 3.1 基本思想
パーツの中身は**テキスト化しない**(実際の漫才に面白さで敵わない & 会話成立が困難なため)。
代わりに **抽象表現**を用いる。

- 表記例: `ボケ【6】(王道)` `ツッコミ【9】(どつき)`
- 各パーツは「**種別** + **威力数値【1〜9】** + **性質タグ**」で表現。
- 威力は **1〜9**(基礎能力の G〜S 表記と視覚的に区別するため、あえて数字)。

### 3.2 パーツの取得(ガチャ方式) ✅
- 「**ネタ作り**」＝**ガチャを1回引く**。**発想力を固定コスト**消費する(週は消費しない・即時)。
- **レア度(威力分布)は基礎能力の合計値が駆動**する。能力が上がるほど高威力が出やすい＝**成長が収集を底上げ**(主人公の成長実感)。
  - 中心威力 c を能力合計100→500で 1.5→7.6 へスライドさせ、ガウス分布(σ=1.6)で抽選。MAXで威力9≈19%・序盤(合計100)はほぼ0%。
  - 能力成長(腕を磨く)も発想力を消費(グレード段階で逓増)。発想力は**収集と成長の唯一の通貨**として奪い合う。
- **パーツはランダムイベント報酬でも入手しうる**(ガチャを唯一の高威力源にしないため、ガチャの高威力はレア寄りでよい)。
- **ゴミ対策**: 低威力パーツが増えたら**ミキサーガチャ**(ボツN枚→追加1回)。パーツ分解→発想力戻し(lossy)は将来案。
- ※グレードは S/A/B/C/D/E/F/G の8段階(G<20 / F20 / E40 / D50 / C60 / B70 / A80 / S90)。
- 数式・確定値の詳細は `research/neta-gacha-design.md`、調整は `research/gacha-tuner.html`。

### 3.3 ネタ構成(打線)
- **9スロット制**(パワプロの打線=9人に準拠)。
- **1スロット目=ツカミ固定、9スロット目=オチ固定**。2〜8番が自由枠。
- 威力数値の合計+コンボボーナスを「**手応え**」として表示(プレイヤーに見える概算期待値。詳細は `SCORING.md`)。
- ※ただし「手応えがあってもウケない」現象を再現するため、本番結果には会場・出順・審査員補正が乗る。

#### 3.3.1 ネタの形式(しゃべくり漫才 / コント漫才) ✅
**ネタを組む際、まず形式を「しゃべくり漫才」か「コント漫才」のどちらかに決める**(パワプロの野手/投手のような大別)。

- **完全分離**: 1つのネタ(打線)は、どちらかの形式で**統一**する。しゃべくり用パーツとコント用パーツは**混在できない**。
- パーツは形式ごとに別物として存在する(例:「しゃべくり用の あるあるボケ」と「コント用の あるあるボケ」は別パーツ)。ネタ作りで集める段階から形式が区別される=ハクスラ的な収集の二系統。
- **形式による能力の活き方**(§2.1):
  - **しゃべくり漫才** → ボケ/ツッコミ/大喜利(共通) + **トーク**(喋りの力が活きる)
  - **コント漫才** → ボケ/ツッコミ/大喜利(共通) + **演技力**(演じる力が活きる)
- これにより、しゃべくり特化ビルド / コント特化ビルドという**構築の二大方針**が生まれる。共通能力(ボケ/ツッコミ/大喜利)は両方で育つので、地続きの成長もある。
- **段階解放(予定)**: ゲーム序盤は**しゃべくり漫才のみ**で進め、ストーリー途中の強制イベントで**コント漫才をアンロック**する方針(初心者の認知負荷を下げ、実装も段階的にするため)。🔧要検討。
- 配置したパーツの性質タグの比率で、ネタの**ジャンルを自動判定**(【王道しゃべくり】【勢いコント】等。形式×タグ傾向で命名)。

### 3.4 パーツ種別と性質タグ ✅
実際の漫才理論(ボケ18種・ツッコミ12種等)を土台に、ゲーム用に厳選。
**形式分離**の有無は種別ごとに異なる(打順固定のツカミ・オチは形式共有、中間スロットのフリ・ボケ・ツッコミはしゃべくり/コント分離)。

#### ツカミ(1番固定・2種・形式共有)
| タグ | 性質 |
|------|------|
| 自己紹介 | 「どうも〜」と名乗って入る王道の掴み。安定型(トーク寄り) |
| ギャグ | 開始直後に一発ギャグをかまして掴む。笑い先行型(ボケ寄り) |

#### フリ(中間・4種・しゃべくり/コント分離)
状況・話題を作り、ボケの前振り(緊張をためる)をする役割。f=形式固有枠(しゃべくり版はトーク/コント版は演技)。
| タグ | 性質 |
|------|------|
| 日常 | 「〜ってあるよね」共感ベースの自然な前振り。あるあるボケと好相性(f依存高め・大喜利低め) |
| 説明 | これから起きることを段取り良く説明し土台を作る。伏線設計・構成型(f・大喜利バランス) |
| 持論 | 「俺はこう思う」独自の主張・理論を展開。理屈型(大喜利寄り) |
| 妄想 | 「もし〜だったら」と空想・仮定から突拍子もない世界を立ち上げる。発想・ひらめき型(大喜利特化) |

> ※「時事」は形式の矛盾(コントの役中で時事を語ると素に戻り不自然)のため通常タグから除外。特殊能力「時事◯」(§2.3)として扱う。

#### ボケ(中間・5種・しゃべくり/コント分離)
| タグ | 性質 |
|------|------|
| あるある | 現実にありそうな共感系。事故りにくい安定型。日常フリと好相性 |
| 誇張 | 大げさ・過剰・行き過ぎ・暴走。パワー型。ハマると倍率大 |
| 裏切り | 期待を外す・スカシ・逆を言う。意外性型。スカシツッコミと好相性 |
| 動き | 体・顔を使うボケ。視覚型。コント形式で特に活きる(演技寄り) |
| ひねり | 言葉遊び・韻・もじり・言い回しの妙。知性型(大喜利・トーク寄り) |

#### ツッコミ(中間・5種・しゃべくり/コント分離)
| タグ | 性質 |
|------|------|
| 直球 | 「なんでやねん」系の王道ストレート。どつき的な強さも含む基本型。安定 |
| 例え | ボケを何かに例えて突っ込む。知性型。大喜利の影響大(「生まれたての子鹿か」系) |
| ノリ | 一度乗ってから突っ込む。テンポ・演技型。直前のボケの笑いを増幅 |
| 説明 | 理詰めで正す・一刀両断。構成・知性型(大喜利寄り) |
| スカシ | 強く突っ込まず流す・はぐらかす。クール・技巧型。裏切りボケと好相性 |

#### オチ(9番固定・2種・形式共有)
| タグ | 性質 |
|------|------|
| 回収 | 前半のフリ・要素を回収して締める。技巧・構成型(大喜利+ボケ寄り)。技巧派審査員に刺さる |
| 裏切り | 最後に予想を裏切る・大きくひっくり返して締める。インパクト型(ボケ寄り)。勢い/新規性派に刺さる |

> **設計メモ**:
> - 各タグは「どの能力をどの比率で参照するか(合計10)」を持つ。共通能力(ボケ/ツッコミ/大喜利)＋形式固有(しゃべくり=トーク/コント=演技)。具体的な配分テーブルと得点計算は `SCORING.md` で定義。
> - 「天丼」「ボケツッコミ」は性質タグではなく**コンボ/特能側**で扱う(§2.3, §3.5)。
> - フリ・ボケ・ツッコミは**しゃべくり版/コント版**を別パーツとして用意。ツカミ・オチは打順固定のため**形式共有**(1タグにつき1つ)。

### 3.5 コンボシステム ✅
特定のパーツ配置で発動する追加得点ボーナス。ネタ構成の戦略性とハクスラ的なやり込みの核。

#### 3.5.1 倍率の基本原則 ✅
- **3段階のレア度**: 小コンボ / 中コンボ / 役満。
- **倍率は威力(基礎点)に乗算**する。リズム判定・能力倍率をかける**前**の段階で効く(`SCORING.md` 参照)。
  - これにより、コンボ込みの「手応え」を編成画面で表示でき、かつコンボもリズムプレイで決めないと実得点に変わらない(本番の緊張感を迂回しない)。
- **倍率値**: 小コンボ **×1.10** / 中コンボ **×1.25** / 役満 **×1.15**。
- **小・中コンボ = 該当パーツのみに厚くかかる(局所)**:
  - 各パーツには、関与する成立コンボのうち**最上位の倍率のみを適用**(小と中は乗算しない)。
  - 例: 中コンボ「教科書」成立箇所では、内包される小コンボ(あるあるネタ・ど真ん中)の×1.10は中の×1.25に負けて不適用。二重がけなし。
  - 別の箇所で成立した下位コンボは併存可。例: 教科書の3パーツは×1.25、別箇所の天丼の2パーツは×1.10。→ ネタの複数箇所にコンボを仕込む戦略性は残る。
- **役満 = ネタ全体(9パーツ)に薄くかかる(全体) + 小・中と両立(重ねがけ)** ✅:
  - 役満成立時、**全9パーツ**に×1.15。さらに小・中コンボとは**両立し乗算**する。
  - 例: 役満かつ中コンボ該当パーツ = 1.15 × 1.25 = 1.4375。役満のみのパーツ(ツカミ等) = 1.15。
  - 小・中(局所・厚い) と 役満(全体・薄い) は役割が異なるため両立。「役満で全体を底上げしつつ、コンボで決め所を爆発」させる重層的戦略。
  - ※役満の発動条件は「中間7枠完全一致」(尻上がり除く)だが、効果は全9枠に及ぶ(中間7枠を極めた結果ネタ全体が輝く、という解釈)。
- **Bad消滅ルール** ✅: コンボ関与パーツが漫才プレイで**Bad判定**になった場合、そのコンボは**不成立**(倍率を付与しない)。Bad以外(Safe/Good/Perfect)なら成立し倍率は満額。
  - ただし倍率満額でも、リズム判定倍率(Perfect1.30〜Safe0.70)は別途各パーツにかかるため、Perfectで決めるほど最終得点は伸びる(コンボ成立のオンオフと、リズム判定の得点差は別レイヤー)。
  - 関与パーツが多いコンボ(役満・4パーツ中コンボ)ほどBad消滅リスクが高い=ハイリスク・ハイリターン。
- **表示と計算の分離** ✅: ネタ構成画面・本番結果では**成立した全コンボ名を表示**(内包小コンボも含む。図鑑にも全コンボ登録)。倍率計算は上記の通り(小中は最上位のみ・役満は全体に両立)。表示は網羅的に・計算は厳密に。
- **最終得点の上限 = 100点** ✅: M-1グランプリがモデルのため100点が満点。倍率計算の結果がどうなっても、最終的に表示・判定に使う得点は100点を超えない(暴走防止の安全装置)。※100点満点スケールへの詳細な換算(威力のスケーリング・閾値の100点換算)は今後の数値バランス調整で詰める。

#### 3.5.2 発動条件のタイプ
- **順序型**: 指定タグが指定の前後関係で並ぶと成立(例: フリ→ボケ)。
- **連続型**: 指定タグが隣接スロットで連続すると成立(間に他を挟まない)。
- **存在型**: 指定タグがネタ内に存在すれば成立(位置・順序は不問。離れていてもよい)。
- **威力差型**: パーツ間の威力差が条件(例: フリとボケの威力差5以上)。

| 7 | 名人芸 | 【しゃべくり】例えツッコミ1＋スカシツッコミ1 | 存在 | なし |
| 8 | 掛け合い | ボケ/ツッコミが交互に4スロット連続 | 連続 | なし |
| 9 | 七変化 | 異なる性質タグのボケが4種類以上 | 存在 | なし |

> マシンガン(同種ボケ連続)と掛け合い(異種交互連続)は好対照=畳みかけの2つの型。
> 熱演・名人芸は形式特化(コント=演技ビルド/しゃべくり=トーク・大喜利ビルド向き)。

#### 3.5.5 役満コンボ(4種) ✅
最もレアで強力。**ネタ全体(9枠)に×1.15**がかかり、小・中コンボと両立(重ねがけ)する(§3.5.1)。最高難度の構成。

| # | 技名 | 発動条件 | 能力条件 | 形式 |
|---|------|----------|----------|------|
| 1 | 尻上がり | 威力が1番→9番にかけて完全右肩上がり(1<2<…<9) | なし | 問わず |
| 2 | 本格しゃべくり | 中間7枠(2-8番)が該当タグで完全一致 | トークB(80)以上 | しゃべくり |
| 3 | 演技派コント | 中間7枠が該当タグで完全一致 | 演技B(80)以上 | コント |
| 4 | 奇想天外 | 中間7枠が該当タグで完全一致 | 大喜利B(80)以上 | 問わず |

**各役満の該当タグ(中間7枠を埋める。フリ2択/ボケ2択/ツッコミ2択)**:
| 役満 | フリ | ボケ | ツッコミ | 質 |
|------|------|------|----------|-----|
| 本格しゃべくり | 日常 or 説明 | あるある or 誇張 | 直球 or ノリ | 二人の掛け合いで魅せる王道(f依存高=トーク映え) |
| 演技派コント | 説明 or 持論 | 動き or あるある | ノリ or スカシ | 役を演じ状況を作るコント(f依存高=演技映え) |
| 奇想天外 | 妄想 or 持論 | ひねり or 裏切り | 例え or スカシ | 発想・ひらめき・意外性(大喜利依存高) |

- **尻上がり**: 完全右肩上がりは低威力パーツも必須となるため、素の手応えが抑制され、役満倍率をかけても得点が青天井にならない設計。配置の妙で魅せる役満。
- 本格しゃべくり/演技派コントは形式が排他のため同一ネタでは同時成立しない。
- 小・中コンボのラインナップ・役満も、ゲーム制作の発展に沿って拡張の余地を残す。

### 3.6 コンボの習得と発見 ✅
コンボ発生順は難易度が高いため、**レシピを知る手段**を用意する。これが知識資産となり周回動機になる。

- **イベントで習得**: 「先輩からの助言」「大物作家からのアドバイス」で骨組みを教わる。相手の格でレシピの格が変わる(同期=小技 / 先輩=大技 / 大物作家・レジェンド=役満)。
- **偶然の発見**: 教わっていないコンボを偶然組んでも**発動する**。ネタ披露後、楽屋で「今日のネタ、〜〜みたいやな」と声をかけられコンボ名が発覚し、**一度披露すれば発見扱い**で図鑑登録。
