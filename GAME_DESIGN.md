## 1. コンセプト / 世界観 ✅

### 1.1 基本コンセプト
パワプロのサクセスの骨格である「ターン制行動選択 → 能力育成 → 定期試験(試合) → 最終評価でキャラ完成 → エンディング」を、野球からお笑い芸人に置き換える。

### 1.2 主人公設定
- **芸歴7年目**の漫才師。吉本的な大手事務所に所属。
- 「**10年目までにM-1風賞レースで優勝できなければ引退**」という約束を背負っている。
- そのため、プレイ期間は **3年間**(7年目→10年目)。

### 1.3 ゴール
- **M-1グランプリ風の賞レースで優勝すること**が最大の目標。
- ただし優勝以外にも多彩なエンディングが用意され、エンド回収が周回動機となる(後述)。

## 2. キャラクター育成パラメータ ✅

### 2.1 基礎能力(5種)
最終的にキャラに固定される、パワプロの「ミート・パワー」等に相当する能力。

| 能力 | 内容 | 対応する漫才パーツ種別 |
|------|------|------------------------|
| **ボケ** | ネタのボケの威力 | ボケ |
| **ツッコミ** | ボケを笑いに変換する力 | ツッコミ |
| **大喜利** | 瞬発的な発想力 | オチ |
| **トーク** | フリートーク・エピソードトーク力 | ツカミ |
| **演技力** | コントの没入感・キャラの説得力 | フリ |

- 評価表記は **G〜S のグレード**(パワプロ準拠)。内部値は 0〜100。
- グレード境界: S(90+) / A(80+) / B(70+) / C(60+) / D(50+) / E(50未満)。

### 2.2 サクセス中のリソースステータス(2種)
育成プレイ中のみ使う、消費・回復するリソース。

| ステータス | 内容 |
|------------|------|
| **体力** | 行動の燃料。本家パワプロと同様の挙動。コマンドで消費、休む/遊びで回復。 |
| **発想力**(オリジナル要素) | ネタ作りに影響。**高いほどネタ作りでレアパーツが出やすい**。インプットで貯め、ネタ作りで消費する「クリエイティブの貯金」。 |

### 2.3 特殊能力(青特/金特/赤特) ✅(体系) / 🔧(全リスト)
パワプロの金特・赤特に相当。本番でエフェクトを発動する。

- **青特(プラス小)**: ◯つかみ / ◯天丼 / ◯スベリ耐性 / ◯客いじり / ◯トップバッター / ◯審査員ウケ / ◯生放送
- **金特(プラス大)**: ◎会場ジャック(爆笑の連鎖) / ◎伝説のネタ(ネタ評価上限突破) / ◎しゃべくりの神
- **赤特(マイナス)**: ✕テンパり / ✕時間オーバー / ✕単独運✕ / ✕審査員と犬猿 / ✕酒癖

> 実装方式は「能力フックシステム」を採用(§4.6 参照)。

## 3. 漫才ネタ構成システム(打線を組む) ✅

### 3.1 基本思想
パーツの中身は**テキスト化しない**(実際の漫才に面白さで敵わない & 会話成立が困難なため)。
代わりに **抽象表現**を用いる。

- 表記例: `ボケ【6】(王道)` `ツッコミ【9】(どつき)`
- 各パーツは「**種別** + **威力数値【1〜9】** + **性質タグ**」で表現。
- 威力は **1〜9**(基礎能力の G〜S 表記と視覚的に区別するため、あえて数字)。

### 3.2 パーツの取得
- 「**ネタ作り**」コマンドやイベントで漫才パーツを入手。
- その時の**発想力が高いほどレアパーツ(高威力・希少タグ)が出やすい**。
- 発想力はネタ作りで消費される。

### 3.3 ネタ構成(打線)
- **9スロット制**(パワプロの打線=9人に準拠)。
- **1スロット目=ツカミ固定、9スロット目=オチ固定**。2〜8番が自由枠。
- 配置したパーツの性質タグの比率で、ネタの**ジャンルを自動判定**(【王道しゃべくり】【シュール漫才】等)。
- 威力数値の合計+コンボボーナスを「**手応え**」として表示。
- ※ただし「手応えがあってもウケない」現象を再現するため、本番結果には会場・出順・審査員補正が乗る。

### 3.4 パーツ種別と性質タグ ✅
各種別5タグずつ(多すぎると構成が難しくなるため)。

**ボケ(5種)**
| タグ | 性質 |
|------|------|
| 王道 | バランス型。どの構成にも馴染み事故りにくい |
| シュール | 高リスク高リターン。会場が冷えているとスベる、ハマると倍率大 |
| 勢い | コンボ向き。連続配置でテンポボーナス |
| 間 | タメ系。リズムゲーでは遅らせタップ。Perfect時の爆発力大 |
| 天丼 | 再利用系。同じ天丼を後半に置くと倍率が乗る |

**ツッコミ(5種)**
| タグ | 性質 |
|------|------|
| どつき | 判定窓が狭いが高倍率の瞬発系 |
| 例え | 単体威力が高い。大喜利力の影響を受ける |
| ノリツッコミ | 直前のボケの笑い量を引き継いで増幅 |
| スカし | シュール系ボケとの相性が良い |
| 共感 | 会場温度を上げやすい安定型 |

**フリ(5種)**
| タグ | 性質 |
|------|------|
| 設定フリ | 世界観を提示。序盤必須級 |
| 日常フリ | 共感系。初期温度を上げる |
| 大ボラフリ | 後のボケの倍率を大きく上げるが、回収失敗時のスベりも大きい |
| テンポフリ | 次パーツの判定窓を広げる補助系 |
| 伏線フリ | オチと対応させると最終倍率アップ |

**ツカミ(5種・案)**: あいさつ系 / 自己紹介いじり / 時事 / 見た目いじり / 変化球
**オチ(5種・案)**: 回収オチ / どんでん返し / 天丼オチ / しりとりオチ(ループ系) / 逃げオチ(低リスク低倍率)

### 3.5 コンボ(役満)システム ✅
特定のパーツ配置で発動する追加得点ボーナス。

- **3段階のレア度**: 小技 / 大技 / 役満。
- 役満コンボの例:
  - 【超シュールネタ】: フリ3連続 → シュールボケ → スカし
  - 【怒涛の畳みかけ】: 勢いボケ+どつき交互4連を6〜8番に配置
  - 【全回収】: 伏線フリ2箇所以上 + 回収オチ
  - 【完全天丼】: 同名の天丼ボケを3・6・8番に配置(間隔ルール)
  - 【教科書】: 王道のみで構成 + 設定フリが2番(低倍率だが事故率ゼロ)
- 小技コンボは2〜3パーツの部分役(例:3連フリ、ノリツッコミ→共感の流れ)を多数散りばめる。

### 3.6 コンボの習得と発見 ✅
コンボ発生順は難易度が高いため、**レシピを知る手段**を用意する。これが知識資産となり周回動機になる。

- **イベントで習得**: 「先輩からの助言」「大物作家からのアドバイス」で骨組みを教わる。相手の格でレシピの格が変わる(同期=小技 / 先輩=大技 / 大物作家・レジェンド=役満)。
- **偶然の発見**: 教わっていないコンボを偶然組んでも**発動する**。ネタ披露後、楽屋で「今日のネタ、〜〜みたいやな」と声をかけられコンボ名が発覚し、**一度披露すれば発見扱い**で図鑑登録。

## 4. 漫才プレイ(リズムゲーム) ✅ 実装済み

### 4.1 基本方式 ─ リズム天国スタイル
- 構成したネタを本番で披露する部分は**リズムゲーム**。
- **BGMのビート自体が譜面**。AudioContext の時計を唯一のマスタークロックとし、楽曲・ノーツ・判定・演出すべてを「拍(beat)」で統一管理 → 音ズレが原理的に起きない設計。
- ルックアヘッド・スケジューラ方式(25ms間隔・150ms先読み)。

### 4.2 操作と判定
- めくり札(パーツカード)が流れ、判定枠「**ココ!**」に重なる瞬間=**音楽のジャスト拍**でタップ。
- 画面のどこをタップしてもOK(スマホの片手操作前提)。
- **4段階判定**: ピタッ!(Perfect) / まずまず(Good) / ズレた(Bad) / …シーン(Miss)。
- 基準判定窓: ±[95, 185, 290]ms(Perfect / Good / Bad)。
- タップすべき拍にはブラス風スタブ和音、1拍前にウッドブロックの予告音 → 耳でも合わせられる。

### 4.3 性質タグ = ゲームプレイ属性 ✅(v6で実装)
タグごとに「札サイズ・判定窓・流れる速度・笑い倍率・固有挙動」が変化し、ネタ構成の戦略性とプレイ難易度を可変にする。

| タグ | 札サイズ | 判定窓倍率 | 移動拍数(小=速) | 笑い倍率 | 固有挙動 |
|------|----------|-----------|------------------|----------|----------|
| あいさつ | M | ×1.05 | 4.0 | ×1.0 | ─ |
| 設定 | L | ×1.20 | 4.0 | ×0.9 | 取りやすい安定枠 |
| 王道 | L | ×1.15 | 4.0 | ×1.0 | ─ |
| どつき | S | ×0.70 | 3.2 | ×1.5 | 速い小札・シビア |
| 大ボラ | M | ×0.95 | 4.0 | ×0.9 | 成功で次のボケ×1.5 / 外すと温度大幅減 |
| シュール | S | ×0.80 | 4.8 | ×1.45 | ゆっくり漂う。温度<40で×0.6 / 温度≥70で×1.3 |
| スカし | S | ×0.85 | 4.0 | ×1.2 | **裏拍**で取る変則リズム。直前シュール成功で×1.3 |
| 間 | M | ×1.10 | 4.0 | ×1.5 | 札が判定枠で停止=休符 → 次の拍でタップ |
| 回収(オチ) | M | ×1.00 | 4.0 | ×1.0 | 温度フィナーレ補正 |
| 勢い | M | ×0.95 | 3.0 | ×1.15 | (拡張) |
| テンポ | M | ×1.00 | 4.0 | ×0.9 | 成功で次札の判定窓拡大(×1.4) |
| ノリ | M | ×1.00 | 4.0 | ×1.0 | 直前の笑い量を50%引き継ぎ |
| 例え | L | ×1.00 | 4.0 | ×1.3 | (拡張) |
| 天丼 | M | ×1.00 | 4.0 | ×1.1 | (拡張) |

- 札サイズ実寸: S=68px / M=82px / L=96px。

### 4.4 会場温度システム ✅
- 初期値30。成功で上昇(Perfect +12 / Good +7 / Bad +2)、Miss で -15。
- ツカミ成功で初期温度に+8ボーナス。
- 温度が高いほど全パーツの笑い倍率が上がる(係数 `0.5 + temp/100`)。
- オチは温度で最終ボーナス(`×(1 + temp/200)`)。
- 客席シルエットが温度に応じて色づき、笑いで跳ねる。
- 🔧 **未反映の改修指示**: 温度の高低でバーの色味を変え、視認性を向上させる。

### 4.5 笑い量の計算式 ✅
```
笑い量 = 威力 × 10
       × 判定倍率(Perfect 1.0 / Good 0.6 / Bad 0.25)
       × タグ笑い倍率
       × 基礎能力補正(対応能力 0〜100 → ×0.9〜×1.1)
       × 温度倍率(0.5 + temp/100)
       + 固有効果(ノリ引き継ぎ / シュール温度補正 / 相性ボーナス 等)
オチのみ最後に ×(1 + temp/200)
```

### 4.6 能力フックシステム(将来アップデート基盤) ✅(基盤実装済み)
- 主人公データ `PLAYER`(基礎能力5種 + 特殊能力リスト)を中心に置く。
  **育成パートはこのオブジェクトを書き換えるだけで本番に反映される**構造。
- **基礎能力補正**: 札の種別 → 対応能力を参照し、判定窓 ±20% / 笑い量 ±10% を常時補正。
- **特殊能力**: `onMiss` / `onPerfect` / `onGameStart` 等の**フック名で登録するデータ駆動方式**。
  条件を満たすと発動し、カットイン演出が出る。
  - 実装済みの例: **◯スベリ耐性**(スベった時の温度低下を半減)。
  - 設計済みフック例: ◎会場ジャック(連続Perfectで温度加算) / ◯つかみ(1番の判定窓+30%) / ✕テンパり(序盤2札の窓-20%)。
- 🔧 **未反映の改修指示**: 能力発動カットインは**画面左側**に出す(視認性が高いため)。

### 4.7 演出 ✅
- **キャラクター**: SVG描画のパワプロ風2頭身(巨大丸頭・点目・眉・ほっぺ・白手袋・白靴・ネクタイ付きスーツ)。ボケ=赤スーツ、ツッコミ=青スーツ。
- **アクション**: ツッコミ時はツッコミがボケを叩き(踏み込み+腕振り上げ)、ボケがのけぞって「ペシッ!」のコミック文字。ボケ時はジャンプ+両手上げ。ツカミ/フリ/オチは二人で弾む。
- **ビート連動**: 毎拍、判定枠が明滅・キャラが小刻みにボブ・観客が左右にスウェイ。画面全体がBPMで脈打つ。
- **カウントイン**: 専用カウントダウンを廃止し、イントロ4〜7拍目に「3・2・1・いくで!」をビートぴったりに表示。
- 🔧 **未反映の改修指示**: Perfectタイミングでウケた時、客の笑い声SEを追加する。

### 4.8 画面レイアウト ✅
- **上部**: 漫才師ステージ(最大領域)。会場温度バー+コンボ数を上端にオーバーレイ。
- **中間**: リズムレーン(めくり札+「ココ!」判定枠)。親指の自然な視線位置。
- **下部**: 観客席(温度で色づき・笑いで跳ねる)+ ウケ量ボード(数字がポンと跳ねる)。

### 4.9 音楽仕様 ✅
- BPM 128 / 4分の4拍子。Aマイナーペンタトニック。
- 構成: スクエア波リード + トライアングルベース + ドラム(キック/スネア/ハイハット)。すべてWebAudioで合成。
- 譜面構造: イントロ2小節(出囃子) → 8拍目からタップ開始 → (間)の休符地帯 → オチでクラッシュ+フィナーレ → アウトロ上昇フレーズ。総演奏時間 約13.6秒(プロト版)。
- 効果音: タップ音 / どつき音(ペチン) / 笑い声(ノイズバースト) / スベり音 / 歓声。

### 4.10 タイミング調整機能 ✅
- タイトル画面に **±150ms のオフセット調整**を搭載(Bluetoothイヤホン等の遅延補正)。

## 5. サクセスパート(育成) ✅

### 5.1 全体構造
- **3年間 = 約150ターン**(1ターン = 1週間)。
- 各年の流れ: **春**(単独ライブ/番組オーディション)→ **夏**(営業・フェス)→ **秋〜冬**(賞レース)。
- 1年目・2年目は賞レース敗退でも続行、**3年目の決勝が最終ゴール**。

### 5.2 行動コマンド(8種)
| コマンド | 効果 |
|----------|------|
| **ネタ作り** | 発想力を消費してパーツ入手。発想力が高いほどレア率UP |
| **ネタ合わせ** | 相方と練習。信頼度UP・基礎能力UP・低ランクコンボ習得(たまに)。**パーツ強化はしない** |
| **インプット** | 映画・読書・街ブラ等。発想力回復のメイン手段 |
| **トレーニング** | 基礎能力を直接強化(筋トレ=体力上限、ボイトレ=ツッコミ、演技レッスン等の選択式) |
| **バイト** | 資金獲得。体力消費。7年目で売れていない悲哀 |
| **遊び** | 体力・メンタル回復。たまに発想力ボーナス |
| **休む** | 体力大回復 |
| ─ | (劇場出番は§5.4の強制イベントとして別建て) |

> **設計判断**: ネタ合わせでパーツ強化を**しない**のは、強化があると序盤パーツの使い回しが可能になり、「より強いパーツを求めてネタ作りを回す」ハクスラ的緊張感が消えるため。

### 5.3 リソース循環
体力(行動の燃料)⇔ 発想力(ネタ作りの燃料)⇔ 資金(レッスン代・機材・交際費)。
基本ループ:「インプットで発想力を貯める → ネタ作りで放出 → 劇場で試す」。

### 5.4 定期劇場出番(8週に1回・強制イベント) ✅
- 選択コマンドとは別建てで「今日は劇場出番だ!」と発生し、漫才(リズムゲーム)をプレイ。
- 150ターン中、約18回の本番。
- 役割: リズムゲーの操作習熟 / ネタの実戦テスト+コンボ発見 / 結果に応じた人気の増減。
- 8週ごとの締切が「次の出番までにネタを仕上げる」目標サイクルを生む。
- **報酬**: 人気の増減 / 出演料(人気が高いほど単価UP) / コンボ発見(楽屋イベント) / 観客の反応データ(ジャンル別の会場相性が見える) / 大ウケ時ボーナス(発想力回復、稀に作家・先輩のコンボ助言イベント)。

### 5.5 評価ゲージ:人気(一本化) ✅
- 評価指標は **人気のみ**に集約。
- 人気が担う役割: 劇場での客の初期温度 / 賞レース会場の声援補正 / イベント発生条件(TV出演解禁等) / エンディング評価。
- 「人気 = この芸人の現在地」を表す総合指標。

### 5.6 相方システム ✅
- ネタ合わせの相手であり、イベントの主軸。
- **信頼度**が本番に影響: 低いとリズムゲーの判定窓が狭くなる(息が合わない)、高いとミスを1回フォロー(◯コンビ愛発動)。
- マイナスイベント(ネタ合わせ中のケンカ等)で信頼度DOWN・体力消費のみ。
- **1年目はマイナスイベント免除**(序盤の理不尽さを消し、「2年目以降に信頼度を疎かにしたツケが回る」学習曲線にする)。
- 喧嘩・解散危機イベントは2年目の停滞期に配置してドラマを作る。

## 6. 賞レース(M-1風) ✅

### 6.1 大会構造(年1回・秋〜冬)
- 1回戦 → 2回戦 → 準々決勝 → 準決勝 → 決勝。
- 7年目の中堅設定のため、1〜2回戦は格下相手のヌルい戦い(チュートリアル的)、準々から本格化。
- 各ラウンドはターン進行に組み込まれ、**ラウンド間でネタを組み替え可能**(戦略の肝)。

### 6.2 決勝の二段構え
- ファーストラウンド(10組)→ 最終決戦(上位3組)。
- **決勝は別ネタ2本必須**(同じネタの使い回し不可、M-1ルール準拠)。
  → 決勝を狙うなら強いパーツを2セット分集める必要がある(リソース配分の悩み)。

### 6.3 本番のランダム要素
- **出順抽選**: 本番直前に抽選。トップバッターは会場初期温度が低い(不利)、後半は温まっているが「前の組が大ウケした直後」はハードルUP。特殊能力◯トップバッターや人気で補正。
- **審査員システム(決勝のみ表示)**: 審査員7人にタイプ設定。技巧派(構成・コンボ評価)/ 勢い派(コンボ数・テンポ)/ 王道派(事故のなさ・完成度)/ 新規性派(シュール系や珍しいコンボに加点)。毎年固定+一部入れ替えで「今年は技巧派が多いから全回収ネタで行く」というメタ読みが成立。
- 準決勝までは審査員非表示の総合点判定でシンプルに。

### 6.4 ライバルコンビ
- 決勝の対戦相手として、固定ライバル2〜3組(シナリオで因縁)+ ランダム生成数組。
- ライバルの得点は乱数で揺れる → 「会心の出来なのに僅差で負ける」M-1の残酷さを再現。

### 6.5 敗退・敗者復活の扱い
- 1〜2年目の敗退はストーリー続行(悔しさイベント → 能力ボーナス)。
- 3年目の敗退はバッドエンド系へ(§7のエンド分岐)。
- **敗者復活戦は演出のみ、プレイヤーは利用不可**。「敗者復活戦結果:◯◯(ライバル)が復活!」と表示され自分は終わる。復活したライバルが決勝で目の前にいる因縁演出にも使える。

## 7. エンディング / エピローグ図鑑 ✅(ロジック) / 🔧(全エンド内容)

### 7.1 コンセプト
パワプロの「サクセス選手のその後」一枚絵を踏襲。**賞レース戦績 × 能力ステータス**の組み合わせで芸人人生のエピローグ(一枚絵 + 説明テキスト)を表示し、**エンディング図鑑に登録**する。図鑑コンプが最大の周回動機。

### 7.2 判定ロジック(確定) ✅
**「上から評価して最初にヒットしたエンドを返す」方式**(レアで豪華なエンドほど上に置く)。

```
1. 隠しフラグを最優先でチェック
   (解散後ピン優勝 / 相方信頼度MAX優勝 等の特殊条件)
   → 該当すれば即その特殊エンドへ

2. 戦績で大分類を決定
   優勝 / 準優勝 / 決勝進出(3位以下) / 準決勝敗退 / それ未満

3. 分類内で能力の突出パターンを上から評価
   複合突出(S/A 2つ以上) → 単一突出(S/A 1つ) → 突出なし高総合 → 低総合

4. 最初に条件を満たしたエンドを採用
```

- **細分化は突出能力重視**(尖り重視。パワプロ的に「ツッコミSで〇〇エンド」と語れる気持ちよさ)。
- **突出の閾値 = 80以上**(A以上を突出扱い)。**Aランク2つでも複合エンドが成立**し、エンド数を稼ぎやすい。
- 突出が複数あれば上位の組み合わせエンドへ。突出ゼロでも総合値が高ければオールラウンダー系へ。

### 7.3 引退回避ライン(確定) ✅
- **「準決勝進出 かつ 総合値が一定以上」は引退エンドを回避**。
- このゾーンは「**賞レースで注目されてバラエティで売れた**」路線のエンディング群とし、**能力値により分岐**させる。
- 総合値 = 基礎5能力の合計(+賞レース戦績ポイントを加算する案あり)で閾値を引く。

### 7.4 エンド例(確定分 + ブレスト案) 🔧
**優勝ルート**
| エンド名 | 条件(例) | 内容 |
|----------|-----------|------|
| 大物MCエンド | 優勝 + ツッコミS + トークS | BIG3の一角。冠番組とCMで頂点へ |
| 唯一無二エンド | 優勝 + ボケS + 大喜利S | 天才と呼ばれ、レジェンド大喜利番組の常連、審査員席へ |
| 怪物芸人エンド | 優勝 + 全能力A以上 | 「優勝はゴールではなかった」。映画監督・世界進出 |
| 演者エンド | 優勝 + 演技S | ピン芸・俳優業へ。連ドラ主演、国際映画祭 |
| 王道スターエンド | 優勝 + 突出なし高水準 | 欠点も尖りもないが国民的存在に |

> 大物MCエンドのサンプルテキスト:
> 「◯◯は優勝後、持ち前の切れ味鋭いツッコミと軽快なトーク力を武器にメキメキと頭角を現し、超売れっ子になった。冠番組やCMを続け、大御所芸人となり、お笑いBIG3の一角を担った。」

**優勝以外のルート(救済を厚く)**
| エンド名 | 条件(例) | 内容 |
|----------|-----------|------|
| 準優勝・伝説エンド | 準優勝 | あの決勝が伝説と語り継がれ、優勝者より売れた |
| バラエティ成功エンド群 | 準決勝進出+総合一定以上 | 注目されてバラエティで売れる。能力で細分化(§7.3) |
| 名脇役エンド | 準決勝止まり + 演技高 | 賞レースには縁がなかったが名脇役に |
| カルト人気エンド | 低戦績 + 特定芸風が極端 | カルト的人気の唯一神 |
| 講師エンド / 家業エンド | 低総合 | 養成所の人気講師に / 実家の家業を継ぎ地元で愛される |

**バッド/特殊エンド**
- 3年目で結果を出せず約束通り引退(相方関係・能力で数パターンに分岐。「廃業したが第二の人生で…」等)。
- 隠しエンド: 解散後ピン優勝「孤高エンド」/ 相方信頼度MAX優勝「最高の相方エンド」 など。

### 7.5 エンド総数の方針 🔧
- まず**戦績5分類 × 主要能力で20種程度のコア**を固め、隠しエンドを後から追加していく方針。
- (図鑑として集めごたえを出すなら最終的に30〜50種を視野)。

## 9. シナリオ(3年間) 🔧 ブレスト段階

### 9.1 年ごとのテーマ(案)
- **1年目「足掻き」**: 基礎固めとシステム解放。マイナスイベント免除。賞レースは準々決勝あたりで現実の壁。ラストに「あと3年」の約束を背負い直す。
- **2年目「迷い」**: 停滞と試行錯誤。芸風を変えるべきか、相方との方向性のズレ、解散危機が集中。賞レースは準決勝が射程。掴んだ手応えが3年目の伏線に。
- **3年目「勝負」**: 集大成。全イベントが回収フェーズへ。これまでの選択でルート分岐。決勝のステージへ。

### 9.2 メインイベント候補(ドラマの幹)
- 相方との解散危機イベント
- ライバルコンビとの直接対決
- 大物作家への弟子入り or 独自路線を貫くかの選択
- テレビ初出演で天狗になる / 潰れるの分岐
- 家族(親の介護・結婚)と芸の二択
- スランプ(ネタが一切ウケなくなる時期)
- 10年目の「約束」を交わした相手(マネージャー? 家族? 元相方?)との因縁
