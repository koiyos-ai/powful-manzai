## 5. サクセスパート(育成) ✅

### 5.1 全体構造
- **3年間 = 約150ターン**(1ターン = 1週間)。
- 各年の流れ: **春**(単独ライブ/番組オーディション)→ **夏**(営業・フェス)→ **秋〜冬**(賞レース)。
- 1年目・2年目は賞レース敗退でも続行、**3年目の決勝が最終ゴール**。

### 5.2 行動コマンド ✅(発想力サイクルに合わせて再設計)
**中心思想**: 「**芸人は舞台・稽古・生活で育つ**」。発想力＝経験(ひらめき)を生活から稼ぎ、ネタ(収集)と腕(能力)に分配する。

**A. 週を消費する行動(稼ぐ・回復・本番)**
| コマンド | 効果 |
|----------|------|
| **ネタ合わせ** | 相方と稽古。**信頼度↑＋コンボ発見が主**(ボケ/ツッコミ自然増)。発想力はオマケ(振れ幅大・たまに大当たり)。**パーツ強化はしない** |
| **インプット** | 映画・読書・観察。**発想力の種供給=主力**(唯一の安定源)＋大喜利自然増 |
| **遊ぶ** | 交流・経験。**専有産出なし=出かけると何かランダムで起きる**(遊び専有イベントあり)。軽い体力回復 |

> **コマンドの差別化(支配的戦略を作らない)**: 各コマンドに**他では代替できない専有産出**を1つ持たせる(ネタ合わせ=信頼/コンボ、インプット=発想力、バイト=資金、休む=体力)。発想力源を**インプット1つに絞る**ことで「最効率の発想力コマンド」が生まれない。**乗算採点**(威力×リズム×能力×コンボ)＋**体力ゲート**により、どれか一つの偏重では総得点が崩れ、全コマンドを回さざるを得ない。遊ぶは専有を持たず「機会(ランダムイベント)」を担う。
| **バイト** | 資金獲得(コンビニ等の固定収入)。体力消費。7年目で売れていない悲哀。**バイト先でランダムイベント発生** |
| **休む** | 体力大回復 |
| (劇場出番) | §5.4の強制イベント。**出来＝内部得点に比例して発想力を獲得** |

> **「営業」はコマンドではなくイベント報酬** ✅: 売れない若手に営業は常時はない。矢田などから「営業の仕事が入ったでげす!」と舞い込む**嬉しい報酬**(資金＋人気＋発想力)。**9〜10月は学祭ラッシュで毎週営業が入る**特別期間。固定の金策はバイト、臨時の旨味は営業、という非対称。

**B. 週を消費しない行動(発想力を使う・即時)**
| コマンド | 効果 |
|----------|------|
| **ネタ作り**(ガチャ) | 発想力を固定コスト消費→パーツ入手(`03_NETA §3.2`)。決勝の別ネタ2本要件と両立させるため**週は消費しない** |
| **腕を磨く**(能力分配) | 貯めた発想力を基礎能力へ分配(パワプロ経験点式・随時)。高グレードほど高コスト＝逓増 |

> **設計判断**:
> - ネタ合わせでパーツ強化を**しない**のは、「より強いパーツを求めてネタ作りを回す」ハクスラ的緊張感を残すため。
> - 旧「トレーニング」(筋トレ=体力上限・ボイトレ=ツッコミ等)は芸人のスキル習得と乖離するため**撤廃**し「腕を磨く」へ。能力は①活動の副産物＋②発想力の分配で伸びる(併用)。
> - ネタ作り・腕を磨きを即時(週消費なし)にしたのは、収集と成長を「週の奪い合い」でなく「発想力の奪い合い」に一本化するため。

### 5.3 リソース循環と発想力サイクル ✅
- リソース: **体力**(週行動の燃料)・**発想力**(創作の燃料＝収集と成長の唯一の通貨)・**資金**(レッスン代・生活費)。
- **発想力サイクル**: 稼ぐ(週消費の生活)→ プールに貯める → 使う(即時・無制限)。使い道は **ネタ作り(パーツ収集)** と **腕を磨く(能力成長)** の2つで、同じ発想力を奪い合う(収集 vs 成長の戦略性)。
- **劇場出番の発想力報酬は内部得点比例**(`base + k×得点`)。上手いプレイ→発想力増→さらに強化、の好循環(baseで下支えしスノーボール抑制)。周回者は入手発想力から決勝点を読める(知識資産)。
- 基本ループ:「稽古/インプット/遊び/本番で発想力を貯める → ネタ作り(収集)と腕を磨く(成長)に配分 → 劇場で試す」。
- 数値設計の詳細・確定値は `research/neta-gacha-design.md`(3年で能力合計100→370着地)。

#### 5.3.1 資金の役割(＝悲哀の軸) 🔧
資金はトレーニング費ではなく、**「売れない芸人の金欠＝悲哀」**を表現する生活の軸とする。
- **定期家賃/生活費**(パワプロ式): 定期で家賃・生活費を支払う。払えないと悲哀ペナルティ(体力/メンタル減等)。難易度は「ちょうどよい」現状を見て**軽め**に。
- **固定の金策＝バイト**、**臨時の旨味＝営業イベント**(§5.2)。
- **資金イベント**(イベント設計で詳細化): 出費(後輩に奢る・機材故障)／臨時収入(営業ヒット)など。
- インプットの本/サブスク代等の細かい固定費は家賃に内包(個別実装はしない)。

### 5.4 定期劇場出番(8週に1回・強制イベント) ✅
- 選択コマンドとは別建てで「今日は劇場出番だ!」と発生し、漫才(リズムゲーム)をプレイ。
- 150ターン中、約18回の本番。
- 役割: リズムゲーの操作習熟 / ネタの実戦テスト+コンボ発見 / 結果に応じた人気の増減。
- 8週ごとの締切が「次の出番までにネタを仕上げる」目標サイクルを生む。
- **報酬**: 人気の増減 / 出演料(人気が高いほど単価UP) / コンボ発見(楽屋イベント) / 観客の反応データ(ジャンル別の会場相性が見える) / 大ウケ時ボーナス(発想力回復、稀に作家・先輩のコンボ助言イベント)。

### 5.5 評価ゲージ:人気(一本化) ✅
- 評価指標は **人気のみ**に集約。
- 人気が担う役割: 劇場での客の初期温度 / 賞レース会場の声援補正 / イベント発生条件(TV出演解禁等) / エンディング評価。
- 「人気 = この芸人の現在地」を表す総合指標。

### 5.6 相方システム ✅
- ネタ合わせの相手であり、イベントの主軸。
- **信頼度**が本番に影響: 低いとリズムゲーの判定窓が狭くなる(息が合わない)、高いとミスを1回フォロー(◯コンビ愛発動)。
- マイナスイベント(ネタ合わせ中のケンカ等)で信頼度DOWN・体力消費のみ。
- **1年目はマイナスイベント免除**(序盤の理不尽さを消し、「2年目以降に信頼度を疎かにしたツケが回る」学習曲線にする)。
- 喧嘩・解散危機イベントは2年目の停滞期に配置してドラマを作る。

## 6. 賞レース(M-1風) ✅

### 6.1 大会構造(年1回・秋〜冬)
- 1回戦 → 2回戦 → 準々決勝 → 準決勝 → 決勝。
- 7年目の中堅設定のため、1〜2回戦は格下相手のヌルい戦い(チュートリアル的)、準々から本格化。
- 各ラウンドはターン進行に組み込まれ、**ラウンド間でネタを組み替え可能**(戦略の肝)。

### 6.2 決勝の二段構え
- ファーストラウンド(10組)→ 最終決戦(上位3組)。
- **決勝は別ネタ2本必須**(同じネタの使い回し不可、M-1ルール準拠)。
  → 決勝を狙うなら強いパーツを2セット分集める必要がある(リソース配分の悩み)。

### 6.3 本番のランダム要素
- **出順抽選**: 本番直前に抽選。トップバッターは会場初期温度が低い(不利)、後半は温まっているが「前の組が大ウケした直後」はハードルUP。特殊能力◯トップバッターや人気で補正。
- **審査員システム(決勝のみ表示)**: 審査員7人にタイプ設定。技巧派(構成・コンボ評価)/ 勢い派(コンボ数・テンポ)/ 王道派(事故のなさ・完成度)/ 新規性派(シュール系や珍しいコンボに加点)。毎年固定+一部入れ替えで「今年は技巧派が多いから全回収ネタで行く」というメタ読みが成立。
- 準決勝までは審査員非表示の総合点判定でシンプルに。

### 6.4 ライバルコンビ
- 決勝の対戦相手として、固定ライバル2〜3組(シナリオで因縁)+ ランダム生成数組。
- ライバルの得点は乱数で揺れる → 「会心の出来なのに僅差で負ける」M-1の残酷さを再現。
- **筆頭ライバル「平成レトロ」**(モデル:令和ロマン)。主人公の同期で、1年目から頭角を現した天才コンビ。詳細は §9.3 を参照。プレイ2年目(芸歴9年目)に優勝し連覇を宣言、3年目(ラストイヤー)の決勝に必ず立ちはだかる最大の壁。

### 6.5 敗退・敗者復活の扱い
- 1〜2年目の敗退はストーリー続行(悔しさイベント → 能力ボーナス)。
- 3年目の敗退はバッドエンド系へ(§7のエンド分岐)。
- **敗者復活戦は演出のみ、プレイヤーは利用不可**。「敗者復活戦結果:◯◯(ライバル)が復活!」と表示され自分は終わる。復活したライバルが決勝で目の前にいる因縁演出にも使える。
