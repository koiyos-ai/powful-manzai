# 矢田 マスターパーツ（原義データ）

ユーザーが手作業で切り抜いたレイヤー素材。**これを原義（基準）データとして管理**し、
着せ替え（表情・動き）はこのパーツの合成で作る。一度作った固定パーツは二度と生成し直さない
＝色味・線のドリフトが起きない。

## 仕様
- **キャンバス：1264×843（全パーツ共通）** … そのまま (0,0) で重ねれば位置が合う。
- **重ね順 (z, 背面→前面)**：`body(L3)` → `arms(L4)` → `head(L1)` → `eyes(L2)`

## パーツ一覧
| ファイル | 層 | 内容 | 役割 |
|---|---|---|---|
| `yada_body.png` | L3 | 胴体（スーツ＋脚＋靴・腕なし・頭なし） | 固定 |
| `yada_head.png` | L1 | 頭＋髪＋素顔（目なし） | 固定 |
| （未）`yada_eyes_<expr>.png` | L2 | 目＋眉＋頬（怒り💢/汗） | **表情で差し替え** |
| （未）`yada_arm_<pose>.png` | L4 | 肩〜腕（左右） | **ポーズで差し替え** |

## 注意
- 元データは JPG（透過が市松模様として焼き込まれ＋ロス圧縮）→ 市松除去して透過PNG化済み。
- **今後のパーツは PNG（本物の透過）で書き出すこと**（JPGはロス劣化＋市松焼き込みのため非推奨）。
- 同一キャンバス・同一スケールを保つこと（位置合わせのため）。
