# 第1部. シナリオ

## 9. シナリオ(3年間) ✅(主要素確定) / 🔧(各年詳細)

### 9.1 年ごとのテーマ
- **1年目「足掻き」**: 基礎固めとシステム解放。マイナスイベント免除。賞レースは準々決勝あたりで現実の壁。ラストに「あと3年」の約束を背負い直す。
- **2年目「迷い」**: 停滞と試行錯誤。賞レースは準決勝が射程。掴んだ手応えが3年目の伏線に。**この年、平成レトロが優勝し連覇を宣言**(§9.3)。主人公コンビに「ラストイヤーにあの壁を越えねば引退」という最悪の状況を突きつける。
- **3年目「勝負」**: 集大成。全イベントが回収フェーズへ。**春に解散危機イベント(§9.3)が強制発生**。これまでの選択(特に信頼度)でルート分岐。決勝のステージで平成レトロと対峙する。

### 9.2 メインイベント候補(ドラマの幹) 🔧
- ✅ **相方との解散危機イベント**(平成レトロ絡み。確定。§9.3 に詳細)
- ✅ **ライバルコンビ「平成レトロ」との因縁・決勝対決**(確定。§9.3)
- 🔧 大物作家への弟子入り or 独自路線を貫くかの選択
- 🔧 テレビ初出演で天狗になる / 潰れるの分岐
- 🔧 家族(親の介護・結婚)と芸の二択
- 🔧 スランプ(ネタが一切ウケなくなる時期)

### 9.3 メインシナリオ詳細:相方・ライバルと解散危機エピソード ✅

#### 9.3.1 相方:矢田(やだ)くん
- **位置づけ**: 主人公の相方であり良き友人。パワプロの矢部くんリスペクトキャラ。
- **出会い**: 養成所の同期。結成から数年が経過(主人公は芸歴7年目)。
- **役割のニュートラル設計**: ボケもツッコミもこなす器用な平均型。主人公がボケ特化なら相方はツッコミに回り、主人公がツッコミ特化なら相方がボケに回る、**主人公の鏡像**。主人公が伸ばさなかった側の能力を最低保証し、尖り特化ビルドの安全網になる。
- **性格**: 人懐っこく憎めないポンコツ愛され。ネタを書くのは主人公なので、主人公が引っ張る関係。一方で**世渡り・処世術に長けたずる賢い一面**を持つ(楽屋で先輩・作家にうまく取り入りコンボ助言を呼び込む、ギャラ交渉や営業の段取りが上手い、審査員の好みを聞き込んでくる等)。「ネタの才能はないが生き残る嗅覚はある」=7年続けられた説得力。
- **持ちギャグ**: 「嫌だ(やだ)」とかけた、すべりがちな自己紹介ギャグ。本人は鉄板のつもりだが客の反応は微妙。会場が温まっている時だけ妙にウケる等、ゲーム内で繰り返し顔を出すお約束ネタ。スベった時の自虐にも使える。
- **感情の伴走者**: ネタがウケれば一緒に喜び、スベれば一緒に落ち込み、賞レースで負ければ悔しがる。プレイヤーの感情の増幅装置。
- **信頼度連動**(§5.6): 高信頼度で本番のミスを1回フォロー(◯コンビ愛)。低信頼度で本番の判定窓が狭くなる(息が合わない)。
- **口調(確定)** ✅: **標準語ベース + 語尾「〜でげす」**(幇間=たいこもち言葉。人に取り入る処世術と、令和の芸人が古風な言葉を使う滑稽さの両面を一語で表現)。一人称は **「オイラ」**。笑いの締めは **「ゲスゲス！」**。主人公の呼称は **「〇〇くん」**(くん付け)。**関西弁にはしない**(関西弁の主人公と被らせず、コンビの喋りを対比させるため)。
  - 自己紹介ギャグ確定形: 「彼女から『帰ってこい』て？　オイラは『や〜だ』。……なんちゃって。やだ、だけにね、ゲスゲス！」
  - **運用方針**: 語尾「でげす」は基本付けるが、全文に付けるとくどいので、感情が高ぶった時・短いツッコミ・喧嘩や真剣な場面では**あえて崩して素を出す**。ギャグキャラが本気になる落差の演出に使う(解散危機の口論では「でげす」を外す等)。
  - **性格イメージ**: 限りなくパワプロの矢部くんと同一。飄々として、お人好しで、憎めない。基本ヘラヘラしているが、ふとした時に主人公を真っ直ぐ気遣う優しさを見せる。主人公をいじりつつ誰より味方。

#### 9.3.2 ライバル:平成レトロ(モデル:令和ロマン)
- **位置づけ**: 主人公の同期コンビ。1年目から頭角を現した天才コンビ。最大の壁(ラスボス)。
- **構成**: **才能あふれるボケ** × **財閥御曹司のツッコミ**。
  - ボケ = 主人公にとって「持っていない才能を持つ男」。同じネタを書く立場として嫉妬と憧れが最も刺さる相手。
  - ツッコミ = 食うに困らない御曹司。ハングリーさのない天才という残酷さの象徴。一方で「親の七光りと言われたくない」等の別種のプライド・葛藤を持たせ、ただの嫌な金持ちにしない。
- **善意ゆえの嫌味**: 天と地ほど差のある主人公コンビにも**気さくに善意で接してくる**。それが無意識に嫌味となり、主人公らは鬱屈を募らせる。「いい奴なのに、いてくれるだけで惨め」という芸人世界の残酷さ。
- **時間設計**: プレイ**2年目(芸歴9年目)で優勝**し、**翌年も出場すると宣言**。これにより3年目(ラストイヤー)決勝に必ず立ちはだかることが2年目時点で確定する=逆算された絶望と高揚。
- **横綱相撲**: 後述の全分岐(最低信頼度エンドを除く)で一貫して高みの見物。「こちらの申し出を断るくらい自信があるなら見せてもらおうか」という余裕。最後まで「いい奴なのに勝てない壁」であり続ける。

#### 9.3.3 解散危機エピソード(3年目・春に強制発生) ✅
**背景**: 平成レトロが事務所を退所し個人事務所で独立。**矢田くんの処世術を買い、マネージャーとして引き抜こうとする**(あくまで善意ベース。「売れない漫才より稼げるよ」というスタンス)。矢田は主人公の「10年目までに結果が出なければ引退」の約束を知っており、平成レトロの優勝+連覇宣言で「10年目での優勝は叶わない」と内心絶望している。

**発端(口論イベント)**: 3年目の春、主人公と矢田のネタ合わせ中に喧嘩が勃発。ラストイヤーのネタ作りの逼迫・焦燥から、主人公が矢田を責めてしまう。言い争いの勢いで、矢田は**マネージャーの話が来ていることを口走ってしまう**。直前に叱責した手前、主人公は強く引き止められず、矢田は「考えさせてほしい」と言い残して去る。

**沈黙期間(ゲームバランス)**: この後、**2週間(調整により3週間も可)、定期の劇場出番(漫才プレイ)が停止**。物語的には「相方がいないので舞台に立てない」表現。ゲーム的には本番がない代わりに**発想力・基礎能力の強化に専念する修行期間**として機能(終盤の最後の能力ブースト)。期間後に強制イベントで和解。

#### 9.3.4 和解イベントの信頼度分岐 ✅
和解の結末は**信頼度**によって分岐する。

| 信頼度 | 結末 |
|--------|------|
| **最低**(※低確率) | **強制ゲームオーバー(バッドエンドに転換)**。矢田はマネージャーへ転身、主人公はピンで活動するも鳴かず飛ばずで中年アルバイター。皮肉にも矢田・平成レトロは事務所を拡大し大御所に。取締役となった矢田からおこぼれの仕事をもらい、芸人として首の皮一枚で生き延びる悲惨な現状エンド。「相方を大事にしないとこうなる」という信頼度システムの存在意義を体現する罰。 |
| **低** | マネージャー話と天秤にかけつつ、**ラストイヤーだけ付き合ってくれる**。「結果が出なければ本当に去る」二重のプレッシャーを背負って決勝へ。 |
| **中** | 主人公が頭を下げ「もう一度やってくれ」と頼む王道の和解。 |
| **高** | **友情イベント発生**。殴り合いの末に仲直り。信頼度がさらに上昇し、**金特◎一心同体を獲得**してラストイヤーへ。 |
| **最高(MAX)** | 高信頼度ルートに加え、**平成レトロも対抗心を燃やして能力アップ**。最高の友情で挑むご褒美が、最高の強敵との真剣勝負になる(最高難度の名勝負)。「楽に勝ちたいなら中信頼度、最高の名勝負がしたいなら最高信頼度」というプレイスタイル選択。 |

> **設計ポイント**:
> - 味方を最高に強化するとラスボスも本気になる、という逆説的な難易度設計。ヌルくならず、やり込みプレイヤーへの敬意になる。
> - 最低信頼度の即ゲームオーバーは**低確率**に留め、初見への脅しとして機能させつつ理不尽な事故死は避ける。
> - 金特◎一心同体は§2.3参照。強力なため通常特殊能力でなく金特イベント取得とする。

#### 9.3.5 ヒロイン:美咲(みさき)※仮名 ✅
- **位置づけ**: 主人公の恋人。**地元・大阪の幼なじみ**。主人公の上京時に大阪へ残った。「10年目までに結果が出なければ引退」という約束に関わる、物語の感情の軸。
- **キャラ造形**: **根はしっかり者だが、おっとりめの大和撫子な王道ヒロイン**。何者でもなかった頃の主人公を知り、その夢を誰より理解している。7年間、健気に大阪で待ち続けている。
- **約束の経緯(確定)**: 期限は**彼女が切るのではなく、主人公側が決意の表れとして提示する**。10年前の上京の日、主人公自らが「てっぺんを獲る」と誓い、その延長として7年目の現在「10年目で結果を出す、ダメなら大阪に帰る」と区切りをつける(§9.5)。
- **彼女システムは実装しない**(ゲームシステム構築の難易度を上げないため。§5の行動コマンドに恋愛育成は含めない)。美咲は育成対象ではなく、**節目に現れる物語装置**として配置(年の区切り・賞レース前後・解散危機など)。主人公の心情を映す鏡。
- **エンディング接続**: 引退エンドでも「大阪に帰って美咲と新しい人生を始める」温かい着地が可能。勝っても負けても物語を受け止める受け皿になる。主人公が芸人を辞める=大阪(彼女のもと)へ帰る、という構造。
- **名前**: 「美咲」は仮名。確定時に差し替え。

### 9.4 メインシナリオ プロット(起承転結) ✅
3年間を貫く一本の物語。確定した幹を時系列に整理。

- **プロローグ(7年目・開幕)**: §9.6 のオープニング脚本に詳細。10年前に夢を抱いて上京した背景、7年経って燻る現実、美咲を待たせ続けている負い目を提示。矢田の一言で重い腰を上げ、**プレイヤーの操作で「10年目までに優勝、ダメなら引退」の決意を確定**させる。約束の重み(過去)とプレイヤーの当事者意識(現在)を両取りする折衷構成。
- **1年目「足掻き」=起**: 基礎固めとシステム解放(チュートリアル的)。賞レースは準々決勝あたりで実力の壁にぶつかり敗退。平成レトロが勝ち上がる姿を横目に最初の鬱屈が芽生える。年末に「残り2年」を噛みしめる。
- **2年目「迷い」=承**: 試行錯誤と最大の衝撃の年。賞レースは準決勝が射程に入り成長を実感。しかし**平成レトロが優勝し連覇を宣言**。手応えを感じ始めた矢先、ラストイヤーの決勝に絶対王者が立ちはだかることが確定する。希望と絶望が同居し、物語が最も揺れる年。矢田も内心、優勝は不可能だと悟り始める(口には出さない)。
- **3年目「勝負」=転**: 集大成。**春に解散危機イベント(§9.3)が強制発生**。重圧と焦燥から主人公が矢田を責め、平成レトロからのマネージャー引き抜きが露見。2〜3週間の沈黙(修行期間)を経て、信頼度に応じた和解イベントで決着。鍛え直した能力と再確認した絆を持って最後の賞レースへ。
- **クライマックス(3年目・決勝)=結**: 勝ち上がった決勝で宣言通り平成レトロと対峙。横綱相撲を貫く彼らに集大成のネタをぶつける。リズムゲーム最大の見せ場。
- **エピローグ(結果分岐)**: 戦績と能力でエンディング分岐(§7の図鑑へ)。優勝/準優勝/バラエティ成功/引退、それぞれの「その後」を紙芝居の一枚絵で描く。

### 9.5 採用した「約束の相手」設計の記録 ✅
- **採用**: 恋人(大阪の幼なじみ・美咲)。
- **約束の時期(確定)**: A案(上京時=芸人人生の出発点で10年の約束)と B案(7年目開幕でプレイヤーが決意)を比較検討し、**折衷案を採用**。すなわち——**約束の決意自体は7年目の開幕シーンでプレイヤーの操作により確定させる(B案の当事者意識)** が、**幼なじみとしての10年分の関係性・上京して離れた歴史はしっかり描く(A案の情緒)**。
- **採用理由**: 「7年目から急に物語が始まる」説得力を最も自然に作れるのが B案寄りの構成だったため(A案は起点の必然性が弱く、自然にしようとするほど B案に接近する)。情緒の厚みは美咲との関係描写で補う。
- **彼女システム不採用の理由**: ゲームシステム構築の難易度を上げないため。美咲は育成対象でなく物語装置とする(§9.3.5)。
- **案5(自分自身への誓い)との両輪**: 美咲と交わした約束=外的な誓いを、7年目の主人公が自分自身の覚悟として引き受け直す、という二層構造を持たせる。

```
【→ チュートリアル / 1年目開始へ】
```

- **構成意図**: モノローグで上京の背景(情緒)を提示→美咲のメッセージで現実を突きつけ→矢田の一言で腰を上げさせ→回想で関係の重みを補強→**プレイヤー操作で決意を確定**(当事者意識)。折衷案の狙い通り両方の強みを順に積む。
- **主人公の名前**: プレイヤー入力(〇〇)。

# 第2部. エンディング / エピローグ図鑑

## 7. エンディング / エピローグ図鑑 ✅(ロジック) / 🔧(全エンド内容)

### 7.1 コンセプト
パワプロの「サクセス選手のその後」一枚絵を踏襲。**賞レース戦績 × 能力ステータス**の組み合わせで芸人人生のエピローグ(一枚絵 + 説明テキスト)を表示し、**エンディング図鑑に登録**する。図鑑コンプが最大の周回動機。

### 7.2 判定ロジック(確定) ✅
**「上から評価して最初にヒットしたエンドを返す」方式**(レアで豪華なエンドほど上に置く)。

```
1. 隠しフラグを最優先でチェック
   (解散後ピン優勝 / 相方信頼度MAX優勝 等の特殊条件)
   → 該当すれば即その特殊エンドへ

2. 戦績で大分類を決定
   優勝 / 準優勝 / 決勝進出(3位以下) / 準決勝敗退 / それ未満

3. 分類内で能力の突出パターンを上から評価
   複合突出(S/A 2つ以上) → 単一突出(S/A 1つ) → 突出なし高総合 → 低総合

4. 最初に条件を満たしたエンドを採用
```

- **細分化は突出能力重視**(尖り重視。パワプロ的に「ツッコミSで〇〇エンド」と語れる気持ちよさ)。
- **突出の閾値 = 80以上**(A以上を突出扱い)。**Aランク2つでも複合エンドが成立**し、エンド数を稼ぎやすい。
- 突出が複数あれば上位の組み合わせエンドへ。突出ゼロでも総合値が高ければオールラウンダー系へ。

### 7.3 引退回避ライン(確定) ✅
- **「準決勝進出 かつ 総合値が一定以上」は引退エンドを回避**。
- このゾーンは「**賞レースで注目されてバラエティで売れた**」路線のエンディング群とし、**能力値により分岐**させる。
- 総合値 = 基礎5能力の合計(+賞レース戦績ポイントを加算する案あり)で閾値を引く。
